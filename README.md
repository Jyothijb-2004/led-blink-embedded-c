#LED blink using Embedded C
# Description
This project demonstates basic LED blinking using Embedded C on 8051 microcontroller.
# Hardware used 
-8051 Microcontroller
-LED
-Keil uvision (for simulation)
#software
-Embedded C
-Keil uvision
# working
The LED connected to port 2 toggle ON and OFF continuously using a delay loop.
